# 샘플 MDX 문서

이것은 MDX 형식으로 작성된 샘플 문서입니다. MDX는 마크다운에 JSX를 추가하여 더 풍부한 콘텐츠를 작성할 수 있게 해줍니다.

## 기본 마크다운 기능

### 텍스트 서식

**굵은 텍스트**와 *기울임 텍스트*를 사용할 수 있습니다. 또한 `인라인 코드`도 표시할 수 있습니다.

### 목록

순서 없는 목록:

- 항목 1
- 항목 2
- 항목 3

순서 있는 목록:

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목

### 인용문

> 이것은 인용문입니다. 다른 사람의 말이나 참고 자료를 인용할 때 사용합니다.
